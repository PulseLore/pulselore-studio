PulseLore Music Meter V1 — Final Web Package

How to publish on Cloudflare Pages:
1. Upload this whole folder as your site, or upload the files inside it.
2. Keep the structure exactly like this:
   index.html
   style.css
   script.js
   assets/pulselore-logo.png
   assets/PulseLore.mp4
3. Open index.html on the deployed website.

Notes:
- Browser-only analyzer.
- No API key.
- No backend.
- Audio files are analyzed locally in the visitor's browser.
- PulseLore.mp4 is used as the hero motion background.
- Logo is used from assets/pulselore-logo.png.

https://pulselore.studio/music-meter/
