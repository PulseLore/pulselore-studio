# PulseLore Studio

Static PulseLore Studio website.

## Run locally

Open `index.html` from the project root, or serve the folder with any static server.

## Hero video

Place the main home video at:

`assets/video/pulselore_hero.mp4`

The home page references it as:

`/assets/video/pulselore_hero.mp4`

If the video is missing, the page falls back to a branded dark PulseLore hero without showing a public error.

## Local-only features

- Music Meter analyzes selected audio locally in the browser.
- Baby QA Garden stores checklist and issues in localStorage.
- Zen Brief Builder stores its last generated brief in localStorage.

## Prototype notes
