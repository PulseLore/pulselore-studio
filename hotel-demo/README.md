# PulseLore Hotel Proposal Page

Upload this entire folder to a hard-to-guess private path such as:

`https://www.pulselore.studio/private/hotel-booking-7k29/`

The folder must contain:

- `index.html`
- `styles.css`
- `proposal.pdf`
- `robots.txt`
- `_headers` (Cloudflare Pages privacy/security headers)

This page intentionally contains no live app demo link. The CTA opens a
prefilled email to `contact@pulselore.studio` requesting a private guided demo.

`noindex` and `robots.txt` reduce search discovery but do not provide password
protection. Anyone with the URL can view or forward the proposal.
